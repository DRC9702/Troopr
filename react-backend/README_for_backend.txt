To run backend:

1 install mongod db
2 in terminal, type:
  mongod
3 in another terminal, in directory : react-backend/ type:
  npm install
  PORT=3001 node bin/www
4 To have your first hardcode data: go to page localhost:3001/addProfile
5 Go to page localhost:3001 to check if a user is returned in json
6 Then you can get your backend data in json from port 3001
